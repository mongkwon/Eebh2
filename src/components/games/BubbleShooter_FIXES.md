# 버블슈터 발견된 문제점 및 수정사항

## 🔴 심각한 문제점들

### 1. 충돌 감지 거리 (726줄)
**문제**: `BUBBLE_RADIUS * 1.8` - 너무 작음
**증상**: 버블이 닿았는데 붙지 않고 통과
**수정**: `BUBBLE_RADIUS * 2.0`으로 변경

### 2. 발사 속도 (1335줄)
**문제**: `speed = 8` - 너무 빠름
**증상**: 프레임 사이에 버블 건너뛰기 가능
**수정**: `speed = 6`으로 감소

### 3. 게임 오버 체크 (1162줄)
**문제**: `canvas.height - 100` vs 버블 y 좌표 (REFERENCE_HEIGHT 기준)
**증상**: 스케일 다르면 게임 오버 잘못 판정
**수정**: `REFERENCE_HEIGHT - 100`으로 변경

### 4. 상태 업데이트 중첩 (1177-1212줄)
**문제**: attachBubble 내부에서 setBubbles 호출 중 setTimeout으로 또 setBubbles 호출
**증상**: 상태 경합, 버블 위치 꼬임
**수정**: 새 줄 추가를 별도 useEffect로 분리 필요

### 5. useCallback 의존성 (1250줄)
**문제**: score가 의존성 배열에 있어서 매번 재생성
**증상**: attachBubbleRef.current 매번 변경, 클로저 문제
**수정**: score 제거하고 함수형 업데이트 사용
